# College-Management-System-
Proper College Management System by using ( OPP |&amp;&amp;| DSA )  in C++ 

## 🚀 Features

- Admin can Handle Both Teachers and Students or New Admissions
- Teachers can add Attendance and Results for Students
- Students can see their Results and Attendance
- All Users has their own Portals with Login Authentication.
- For Default Passwords See and Open portals.h

## 🛠️ Tech Stack used and How to Deploy

- Save all files in one folder in same order which is in repo.
- .cpp file is the main file run it on your compiper after all other files was on same folder as .cpp

- OOP and DSA is used , major focused data structure is Queue and Link List
- All data is saved as Xcel/CSV files using C++ FILE HANDLING.
